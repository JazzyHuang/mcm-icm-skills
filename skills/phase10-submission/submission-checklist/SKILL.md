# 提交检查清单 (Submission Checklist)

## 功能概述

生成完整的提交检查清单。

## 检查清单

```markdown
## MCM/ICM 2026 提交检查清单

### 文件要求
- [ ] PDF文件名为控制号 (如: 2412345.pdf)
- [ ] 文件大小 < 20MB
- [ ] PDF可正常打开

### 页面要求
- [ ] 总页数 ≤ 25页
- [ ] 摘要页为第一页
- [ ] 字体 ≥ 12pt

### 匿名要求
- [ ] 无学校名称
- [ ] 无团队成员姓名
- [ ] 无指导教师姓名

### 标识要求
- [ ] 控制号正确显示
- [ ] 选题标识正确 (A/B/C/D/E/F)

### 内容要求
- [ ] 摘要完整 (300-500词)
- [ ] 所有章节完整
- [ ] 参考文献格式统一
- [ ] 图表编号连续

### 时间要求
- [ ] 提交截止: 2026年2月2日 21:00 EST
- [ ] 留出提前量

### 提交步骤
1. 登录 contest.comap.com
2. 输入Advisor ID和密码
3. 选择题目
4. 上传PDF文件
5. 确认提交
6. 保存确认页面截图
```

## 输出格式

```json
{
  "checklist": {
    "total_items": 18,
    "completed": 18,
    "pending": 0,
    "deadline": "2026-02-02T21:00:00-05:00",
    "status": "Ready for submission"
  }
}
```

## 重要提醒

- **提前提交**: 建议提前2-4小时提交，避免网络拥堵
- **保存确认**: 提交后务必保存确认页面截图
- **多次检查**: 提交前至少检查两遍

## 相关技能

- `pre-submission-validator` - 提交前验证
- `submission-preparer` - 提交准备
