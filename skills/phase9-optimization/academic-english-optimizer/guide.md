# 学术英语优化器 (Academic English Optimizer)

## 功能概述

专项优化论文的学术英语质量。

## 优化方向

### 1. 精确性 (Precision)
- 使用精确词汇
- 避免模糊表达
- 量化描述

### 2. 客观性 (Objectivity)
- 避免主观判断
- 使用中性语言
- 数据支撑论点

### 3. 清晰度 (Clarity)
- 简洁句式
- 逻辑清晰
- 避免歧义

### 4. 一致性 (Consistency)
- 术语统一
- 风格统一
- 格式统一

## 优化示例

| 原文 | 优化后 |
|------|--------|
| very big | substantial/significant |
| quite good | effective/efficient |
| We think that | Evidence suggests that |
| It is obvious that | The data indicate that |

## 输出格式

```json
{
  "optimization": {
    "total_improvements": 38,
    "categories": {
      "precision": 12,
      "objectivity": 8,
      "clarity": 10,
      "consistency": 8
    }
  }
}
```
