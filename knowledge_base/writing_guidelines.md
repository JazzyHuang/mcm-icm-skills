# MCM/ICM 学术写作指南 (增强版)

## 一、总体原则

### 1. 精确性 (Precision)
- 使用精确的词汇和表达
- 避免模糊的描述如"很多"、"大约"
- 尽可能量化结果

### 2. 客观性 (Objectivity)
- 使用中性语言
- 避免主观判断
- 用数据支撑论点

### 3. 清晰度 (Clarity)
- 句子简洁明了
- 逻辑结构清晰
- 避免歧义

### 4. 一致性 (Consistency)
- 术语使用统一
- 符号定义一致
- 格式规范统一

## 二、时态使用

### 一般现在时
- 描述普遍真理
- 描述方法和模型
- 引用文献观点

```
The model assumes that... (模型假设...)
This approach minimizes... (这种方法最小化...)
```

### 一般过去时
- 描述实验过程
- 描述已完成的工作
- 描述数据收集

```
We collected data from... (我们收集了数据...)
The experiment showed that... (实验表明...)
```

### 现在完成时
- 描述持续影响的工作
- 强调与当前的关联

```
Previous studies have shown... (先前研究表明...)
```

## 三、避免使用

### 1. 口语化表达
❌ "a lot of" → ✅ "numerous", "substantial"
❌ "big" → ✅ "significant", "considerable"
❌ "good" → ✅ "effective", "efficient"
❌ "bad" → ✅ "poor", "ineffective"

### 2. 主观判断
❌ "We think that..." → ✅ "Evidence suggests that..."
❌ "Obviously,..." → ✅ "The data indicate that..."
❌ "It is clear that..." → ✅ "Results demonstrate that..."

### 3. 不推荐的词汇
❌ Firstly, Secondly (用First, Second)
❌ In order to (用To)
❌ Due to the fact that (用Because)
❌ At the present time (用Currently)

### 4. 感叹号
永远不要在学术论文中使用感叹号！

## 四、学术词汇

### 描述趋势
- increase, rise, grow (上升)
- decrease, decline, drop (下降)
- fluctuate (波动)
- remain stable (保持稳定)

### 描述程度
- significantly (显著地)
- substantially (实质性地)
- marginally (略微地)
- dramatically (剧烈地)

### 描述关系
- correlate with (与...相关)
- depend on (依赖于)
- influence (影响)
- contribute to (贡献于)

### 描述方法
- propose (提出)
- develop (开发)
- implement (实现)
- validate (验证)

## 五、段落结构

### 标准段落结构
1. **主题句** - 段落核心观点
2. **支撑句** - 证据和解释
3. **过渡句** - 连接下一段落

### 示例
```
[主题句] Our model demonstrates superior performance compared to 
existing methods. [支撑句] Specifically, the RMSE decreased by 23% 
while computational time was reduced by half. [解释] This improvement 
can be attributed to the novel hybrid architecture that combines 
physical constraints with data-driven learning. [过渡句] Building 
on these results, we further analyzed the sensitivity of key parameters.
```

## 六、引用规范

### 引用格式
- 使用期刊要求的引用格式
- 美赛常用：数字引用 [1] 或 作者-年份 (Smith, 2024)

### 引用时机
- 陈述非原创观点时
- 使用他人数据或方法时
- 对比前人工作时

### 引用技巧
```
According to Smith et al. [1], ... (根据Smith等人...)
Previous studies [1,2] have shown that... (先前研究表明...)
This approach was first proposed by... (这种方法首先由...提出)
```

## 七、O奖写作特点

### 1. 创新点突出
- 明确说明创新之处
- 与现有方法对比
- 量化改进效果

### 2. 逻辑严谨
- 每个假设都有论证
- 每个结论都有支撑
- 因果关系清晰

### 3. 表述专业
- 使用领域术语
- 数学表达规范
- 图表专业美观

### 4. 自我批评
- 诚实承认局限性
- 提出改进方向
- 展示学术成熟度

## 八、常见中式英语错误及修正 (新增)

中式英语(Chinglish)是影响论文质量的重要因素。以下是最常见的错误模式和修正建议：

### 8.1 陈词滥调开头 (严重程度: 高)

| 错误表达 | 正确表达 | 说明 |
|---------|---------|------|
| With the development of... | As X advances/evolves | 过度使用的陈词滥调 |
| With the rapid development of... | Recent advances in X | 同上 |
| In recent years... | Recently / Over the past decade | 不够精确 |
| Nowadays... | Currently / At present | 口语化 |
| As we all know... | [直接陈述事实] | 假设读者知识 |
| It is well known that... | [直接陈述事实] | 无意义填充 |
| As is known to all... | [直接陈述事实] | 同上 |
| Under the background of... | In the context of / Given | 直译 |
| With the coming of... | The emergence of / The advent of | 直译 |
| Along with the progress of... | As X progresses | 直译 |

### 8.2 程度表达 (严重程度: 中)

| 错误表达 | 正确表达 | 说明 |
|---------|---------|------|
| more and more | increasingly | 口语化 |
| more and more important | increasingly important | 同上 |
| very very | extremely / highly | 重复强调 |
| a lot of | numerous / substantial / many | 口语化 |
| lots of | numerous / many | 口语化 |
| big/small | large/small, significant/minor | 口语化 |
| good/bad | effective/ineffective | 口语化 |
| get better/worse | improve/deteriorate | 口语化 |
| very unique | unique | 逻辑错误 |

### 8.3 动词搭配 (严重程度: 中)

| 错误表达 | 正确表达 | 说明 |
|---------|---------|------|
| plays an important role | serves as / functions as / is essential for | 过度使用 |
| has great influence on | significantly affects/influences | 更直接 |
| make a contribution to | contribute to | 名词化冗余 |
| give a description of | describe | 名词化冗余 |
| make an analysis of | analyze | 名词化冗余 |
| conduct research on | research / investigate | 更简洁 |
| put forward | propose / present / suggest | 直译 |
| solve the problem | address/tackle the problem | 更学术 |

### 8.4 冗余表达 (严重程度: 低)

| 错误表达 | 正确表达 | 说明 |
|---------|---------|------|
| basic fundamentals | fundamentals | 同义重复 |
| future prospects | prospects | 同义重复 |
| past history | history | 同义重复 |
| true fact | fact | 同义重复 |
| final conclusion | conclusion | 同义重复 |
| completely eliminate | eliminate | 同义重复 |
| absolutely essential | essential | 同义重复 |
| end result | result | 同义重复 |

### 8.5 句式结构 (严重程度: 中)

| 错误表达 | 正确表达 | 说明 |
|---------|---------|------|
| The reason is because... | The reason is that... / Because... | 重复 |
| It can be seen that... | [直接陈述] / Clearly, | 冗余开头 |
| It should be noted that... | Notably, / Note that | 更简洁 |
| There is no doubt that... | Clearly, / Undoubtedly, | 更简洁 |
| In order to... | To... | 更简洁 |
| So as to... | To... | 更简洁 |
| For the purpose of... | To / For | 冗余 |
| Due to the fact that... | Because / Since | 冗余 |
| On the basis of... | Based on | 更简洁 |
| By means of... | By / Through / Using | 更简洁 |

### 8.6 学术写作特有 (严重程度: 中-高)

| 错误表达 | 正确表达 | 说明 |
|---------|---------|------|
| This paper will discuss... | This paper discusses... | 时态问题 |
| In this paper, we will... | In this paper, we... | 时态问题 |
| Through this study... | This study... / In this study, | 冗余 |
| By using the method... | Using the method... | 冗余 |
| The experimental results show that... | Results indicate/show that... | 冗余 |
| After careful analysis... | Analysis reveals... | 主观表达 |
| We think/believe that... | Evidence suggests that... | 主观表达 |
| It is proved that... | This demonstrates/indicates that... | 过强声明 |
| It is certain that... | This suggests/indicates that... | 过强声明 |

## 九、数学符号表达规范 (新增)

### 9.1 符号定义规范

```latex
% 好的定义方式
Let $n$ denote the number of samples.
We define $\alpha$ as the learning rate.
The variable $x_i$ represents the $i$-th feature.

% 避免的方式
$n$ is the number of samples.  % 缺少"Let"或"denote"
```

### 9.2 向量与矩阵表示

| 类型 | 表示方式 | 示例 |
|------|---------|------|
| 标量 | 斜体小写 | $x$, $\alpha$ |
| 向量 | 粗体小写 | $\mathbf{x}$, $\boldsymbol{\theta}$ |
| 矩阵 | 粗体大写 | $\mathbf{A}$, $\mathbf{X}$ |
| 集合 | 花体/粗体大写 | $\mathcal{S}$, $\mathbb{R}$ |

### 9.3 公式编号规范

- 所有重要公式应编号
- 被引用的公式必须编号
- 编号应按章节顺序，如 (1), (2) 或 (3.1), (3.2)
- 引用时使用 "Equation (1)" 或 "Eq. (1)"

### 9.4 单位表示

```latex
% 正确
The temperature is $25~\mathrm{°C}$.
The speed is $10~\mathrm{m/s}$.
The area is $100~\mathrm{km}^2$.

% 避免
The temperature is 25°C.  % 数字和单位之间应有空格
The speed is 10 m/s.     % 单位应为正体
```

## 十、Hook句设计指南 (新增)

摘要的开头句(Hook)是评委第一印象的关键。

### 10.1 三种Hook模式

**模式1: 问题重要性钩子**
```
Climate change threatens to displace over 200 million people by 2050, 
yet current models fail to capture migration dynamics.
```

**模式2: 数据冲击钩子**
```
Our analysis reveals that a mere 3.7% adjustment in traffic signal timing 
can reduce urban congestion by up to 28%.
```

**模式3: 方法创新钩子**
```
While traditional approaches treat evacuation as a static problem, 
we reconceptualize it as a dynamic game between evacuees and hazards.
```

### 10.2 Hook检查清单

- [ ] 是否在5秒内抓住注意力？
- [ ] 是否包含具体数字？
- [ ] 是否避免了陈词滥调？
- [ ] 是否在25-35词之间？
- [ ] 是否自然引出研究问题？

### 10.3 Hook禁忌

```
❌ "With the development of artificial intelligence..."
❌ "In recent years, X has attracted widespread attention..."
❌ "It is well known that..."
❌ "Nowadays, the problem of X becomes more and more serious..."
```

## 十一、量化表达指南 (新增)

### 11.1 数字精确度

| 情况 | 建议精度 | 示例 |
|------|---------|------|
| 百分比改进 | 1-2位小数 | 23.5%, 不是23.456% |
| 误差指标 | 3-4位有效数字 | 0.0312, 不是0.031234567 |
| 大数字 | 使用科学计数法 | $1.2 \times 10^6$, 不是1200000 |
| 置信区间 | 2位小数 | [0.82, 0.88] |

### 11.2 量化词汇替换

| 模糊表达 | 量化表达 |
|---------|---------|
| significantly improved | improved by 23% |
| much better | 2.5 times faster |
| relatively high | above 0.85 |
| most of | 78% of |
| around/about | 具体数值 + 置信区间 |

### 11.3 O奖论文量化密度

O奖论文摘要通常包含4-8个具体数字，包括：
- 1-2个问题规模数字（影响人数、经济损失等）
- 2-3个结果数字（精度、改进幅度等）
- 1-2个方法参数（模型维度、训练数据量等）
